# HabitForge Frontend (Vite)

See the repo root `README.md` for the full setup (Postgres, backend migrations, and running everything together).

## Commands

Install:

```bash
npm install
```

Dev server (port `5174`):

```bash
npm run dev
```

Build:

```bash
npm run build
```

Preview (port `4173`):

```bash
npm run preview
```
